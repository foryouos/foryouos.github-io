# 银行管理系统

开源注释详解:Github

# 功能

  • 添加账户
    • 储蓄账户
    • 信用卡账户
  • 存款
  • 取款
  • 当前账户余额
  • 指定时间内的账目信息
    • 利息
  • 本地化存储，二次启动同步
  • 所有账户总金额
  • 退出

# 数据存储思路

将每次运行的代码存储到本地 txt 文件当中，每次启动可以首先初始化文件内容，已达到文件储存的目的

# 缺点

  • 数据量过大数据初始化难
  • 业务逻辑越复杂也越难以保存
  • 用户交互需要按照文件保存书写指令，十分不便

或许可以使用 MySQL 数据库存储数据，用 C++ 与 MySQL 进行交互，达到相关查询，存储的功能。

# 各文件功能

  • date.h 日期类头文件
  • date.cpp 日期类实现文件
  • account.h 账户类定义头文件
  • account.cpp 账户类实现文件以及记录

  • main.cpp 主函数文件
