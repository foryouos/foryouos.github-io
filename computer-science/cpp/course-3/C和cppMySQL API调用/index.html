# MYSQL_ROW
typedef char** MYSQL_ROW;
  • 返回值二级指针， char** 指向一个指针数组，类型是数组，里面的元素仍未指针， char* 类型，
  • char* [] ：数组中的字符串对应的一列数据
  • 如果想要遍历整个结果集，需要对函数 MYSQL_ROW 进行循环调用
