# Github 自定义域名

  • 进入主项目的 setting 设置个人的项目名 id.github.io, 在左边的 page 下设置个人域名，在并强制 HTTPS

  • 在域名服务端设置（以百度云为例)
    除了第五个记录值改为自己的其它完全相同

# https 问题:

  • 若设置了 Https，Chrome 浏览器网址左边的没有小黑锁，显示不安全，点 F12 进入开发者选项，看那个链接没有 https，修改即可实现小黑锁，安全。
